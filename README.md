This React Cryptocurrency Api uses API to fetch all the Cryptocurrencies and exchanges and another API to display the latest news on the subject. The app is deployed on [Netlify](https://www.netlify.com/).

## Create React App

In the terminal type **npx create-react-app crypto-app** and click enter. This will create your React app.

**Install** all the **dependencies** for the project with:  
npm install antd @ant-design/icons react-redux @reduxjs/toolkit axios chart.js html-react-parser millify moment react-chartjs-2 react-router-dom

## Technologies Used:

- [Reactjs](https://reactjs.org/)
- [Chartjs](https://www.chartjs.org/)
- [Redux](https://redux-toolkit.js.org/)
- [Rapid api](https://rapidapi.com/hub?utm_source=youtube.com%2FJavaScriptMastery&utm_medium=DevRel&utm_campaign=DevRel)
- [Bing News Search API](https://rapidapi.com/microsoft-azure-org-microsoft-cognitive-services/api/bing-news-search1/)
- [Ant Design](https://ant.design/)

## Deploy to Netlify

1. In your terminal type **npm run build**, this will create a build directory with a production build of your app.
2. Go to Netlify and **signin**/signup.
3. Click on **'Sites'** and then you can drag and drop your build folder to deploy your site.
